# sudokju

Sudoku solver implemented in Clojure.

## Usage

    $ java -jar sudokju-0.1.0-standalone.jar [puzzle.json]

## Puzzle format
```json
[
  [ 3, 0, 4, 0 ],
  [ 0, 1, 0, 3 ],
  [ 2, 3, 0, 0 ],
  [ 1, 0, 0, 2 ]
]
```
